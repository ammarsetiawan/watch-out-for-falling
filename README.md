## ⚡ Apa Itu "Watch Out for Falling"?
Uji refleksmu di dunia yang sedang runtuh! **Watch Out for Falling** adalah game bertempo cepat di mana satu-satunya tujuanmu adalah bertahan hidup dari musuh musuh yang mencoba menangkap mu. Semakin lama kamu bertahan, semakin gila tantangannya!

> "Mudah dimainkan, sulit untuk dikuasai."

---

## 🚀 Cara Memulai (Instalasi Cepat)

Tidak perlu instalasi yang rumit! Kamu bisa langsung terjun ke dalam aksi dengan langkah berikut:

1.  **Download** atau **Clone** repositori ini.
2.  Buka folder hasil download.
3.  Cari dan klik dua kali pada file:
    ### 📂 `watch out for falling.exe`
4.  Selamat bersenang-senang!

## 🕹️ Kontrol Game
Gunakan kombinasi tombol berikut untuk mendominasi papan skor:

| Tombol | Aksi |
| :--- | :--- |
| `←` / `A` | Gerak Kiri |
| `→` / `D` | Gerak Kanan |
| `Esc` | start Game |
